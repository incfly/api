Services

MeshTroubleshootingService

MeshTroubleshootingService is the mesh wide trouble shooting API.

rpc GetConfigDump(GetConfigDumpRequest) returns (GetConfigDumpResponse)

GetConfigDump gets the config dump of the proxy.

rpc SetLogLevel(SetLogLevelRequest) returns (SetLogLevelResponse)

Set the proxy log level.

ProxyTroubleshootingService

ProxyTroubleshootingService is exposed by the proxy management component on each node. Currently this is pilot.

rpc Troubleshoot(TroubleshootResponse) returns (TroubleshootRequest)

TroubleShoot request on proxy level.
